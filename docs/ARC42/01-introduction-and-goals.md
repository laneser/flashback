# 1. 簡介與目標

## 1.1 需求概述

Flashback 是一套**飆車聲音偵測與閃光嚇阻系統**，運行於 Android 手機上（iOS 版本計畫中）。

**核心問題：** 台灣都市深夜飆車問題嚴重，改裝車輛炸街聲音超過 100dB，嚴重影響居民生活品質。現有處理方式（檢舉、警察巡邏）效率低且缺乏即時性。

**解決方案：** 利用手機麥克風持續監聽環境音，透過 AI（YAMNet）辨識飆車聲音特徵，自動觸發閃光燈模擬測速照相，利用飆車族對「被拍照」的心理恐懼達到嚇阻效果。

## 1.2 品質目標

| 優先級 | 品質屬性 | 目標 |
|--------|---------|------|
| 1 | 即時性（Latency） | 從偵測到閃光觸發 < 100ms（本機） |
| 2 | 準確率（Accuracy） | 飆車聲音辨識率 > 90% |
| 3 | 低誤觸發率（Precision） | 誤觸發率 < 5%（正常車流、喇叭聲等不觸發） |
| 4 | 可用性（Availability） | 支援長時間（8+ 小時）連續運作 |
| 5 | 易用性（Usability） | 安裝至觸發僅需基本設定，非技術人員可操作 |

## 1.3 利害關係人

| 角色 | 期望 |
|------|------|
| 社區居民（使用者） | 簡單設定即可使用，有效嚇阻飆車噪音 |
| 開發者 | 清晰的架構、模組化設計，方便貢獻 |
| 法律顧問 | 系統不侵犯隱私、不造成交通安全風險 |
| 開源社群 | 可自行架設、擴充，形成分散式嚇阻網絡 |
